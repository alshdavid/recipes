# James Brownie

## Ingredients

| Ingredient        | Quantity |
| ----------------- | -------- |
| Sugar             | 2 Cups   |
| Flour                 | 1 Cup    |
| Coca Powder           | 2/3 cup  |
| Chocolate (bits 40%)  | 200g     |
| Butter            | 200g     |
| Vanilla           | 2tsb+    |
| Eggs              | 4        |

## Method

1. Preheat oven to 170C (grill mid rack)
2. Place brownies in tray
3. Place tray in oven for 20 - 25 minutes
4. Take tray out
5. Let cool
