# Pumpkin Soup

## Ingredients

| Ingredient        | Quantity                    |
| ----------------- | --------------------------- | 
| Butternut Pumpkin | 1 Whole (aprox 800g)        |
| Potatoes          | 2 Medium Sized (aprox 330g) |
| Onion             | 1                           |
| Red Lentils (dry) | 1 cup (160g)                |
| Water             | 2 cups (500ml)              |

## Method

1. Chop all the things
2. Pressure cook for 30 minutes
3. Blend all the things
